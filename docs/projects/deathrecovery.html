Death Recovery Documentation

AP: Death Recovery Mod

AP: Death Recovery adds a Recovery Stone (unlocked at level 25) that allows players to recover items from their latest death! Customization options are available using a companion mod, AP: Mod Menu.

How to Use

Using the mod is simple; just find the Recovery Stone at level 25, place it down, and interact with it to recover your items!

Recovery Stone Spawn Code

cheat gfi PrimalItemStructure_Grave_APDR 1 0 0
